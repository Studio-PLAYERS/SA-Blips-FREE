# SA-Blips-FREE
This is 1st version of Blip creator. Basic script for creating blips. You can easily create blip in config.lua

```
Config = {}

Config.Locations = {
    -- Blip mechanik
    ["pdm"] = {
        ["label"] = "Premium Deluxe Motorsport",
        ["coords"] = vector4(-57.0654, -1097.0039, 26.4224, 205.0292),
        ["showblip"] = true,
        ["scale"] = 0.8,
        ["blipsprite"] = 523,
        ["blipcolor"] = 11
    },
```
